# To Compile Cpp Files Seprate
* `cd` to the driectory using command line
* use `clang++ -std=c++11 file_name.cpp`
* type `./executable_name`

# To use Make File
* `cd` to the directory using command line
* type `make`
* watch as it compiles
* type `./execuable_name`
